# Jenu-Gumpu 
### Smart Honey Collection and Management Application

Jenu-Gumpu is a modern Android application developed using **Kotlin** and **Jetpack Compose** to support honey collection communities and
beekeepers through digital management workflows. The application provides features such as harvest tracking, stock monitoring, honey 
grading, market price visualization, multilingual support, and profit calculation systems.

---

##  Project Overview

The project was developed as part of the **MindMatrix Android Application Development Internship Program**.  
It focuses on building a responsive and user-friendly Android application using modern Android development practices and AI-assisted 
workflows.

The application helps honey collection groups digitally manage:
- Harvest records
- Honey stock
- Market price tracking
- Profit estimation
- Honey quality grading
- Multilingual communication


2. Open the project in Android Studio

3. Sync Gradle files

4. Run the application using:
   - Android Emulator
   - Physical Android Device
